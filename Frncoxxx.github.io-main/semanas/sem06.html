    SEMANA 6: Componentes en React

    Esta semana aprendimos la base de React: cómo crear y renderizar componentes y cómo usar JSX para combinar HTML con JavaScript. También vimos las formas de aplicar estilos dentro de una aplicación React.

    Lo que vimos en clase

    • Qué es ReactDOM.render() y cómo React actualiza el DOM sin recargar la página.
    • Cómo crear componentes de clase y componentes funcionales.
    • Cómo insertar variables y expresiones en JSX.
    • Uso de condicionales (if, ? :, &&) y bucles (map()) dentro de JSX.
    • Diferentes formas de aplicar estilos: archivos .css, módulos CSS e inline styles.

    Ejercicios de Laboratorio

    Durante el laboratorio se aplicaron los conceptos teóricos de la creación de componentes en React. Se desarrollaron ejercicios prácticos donde se construyó una aplicación con siete componentes y diseño responsivo en CSS.
    También se trabajaron componentes anidados, comunicación entre componentes (padre e hijo, hermanos) y el renderizado de datos en tablas utilizando la función .map().
    Estas actividades permitieron comprender la estructura modular de React y cómo organizar una aplicación dividiéndola en partes reutilizables.
