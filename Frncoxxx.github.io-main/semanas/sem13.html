SEMANA 13: Desarrollo Backend PHP con Laravel

En esta semana se desarrollaron aplicaciones web del lado servidor utilizando PHP y el framework Laravel. Se revisó el funcionamiento del servidor Apache junto con PHP, el proceso de interpretación del código PHP y su conversión a HTML, además de la estructura moderna de desarrollo que ofrece Laravel bajo el patrón MVC (Modelo–Vista–Controlador).

Lo que vimos en clase

• Funcionamiento del servidor Apache HTTP Server y ejecución de PHP.
• Instalación de PHP desde archivos ZIP y configuración del archivo php.ini.
• Uso del archivo httpd.conf para configurar Apache (DocumentRoot, Listen, ServerName).
• Instalación y uso de Composer como gestor de dependencias en PHP.
• Sintaxis básica de PHP: variables, condicionales, bucles y funciones.
• Introducción al framework Laravel y su estructura de directorios.

Conceptos clave de PHP
Elemento Descripción Ejemplo
Variables Se declaran con el signo $ $nombre = "Ana";
Condicionales Controlan el flujo del programa if ($edad >= 18)
Bucles Repiten bloques de código for, while, foreach
Funciones Permiten reutilizar lógica function saludar()
Introducción a Laravel

Laravel es un framework de PHP de código abierto que facilita el desarrollo de aplicaciones web modernas. Se basa en el patrón MVC y ofrece herramientas como rutas, controladores, vistas Blade, Eloquent ORM, middleware y Artisan para automatizar tareas comunes del desarrollo.

Ejercicio de Laboratorio

En el laboratorio se desarrolló una aplicación web en Laravel para registrar estudiantes. Las principales actividades fueron:

• Creación de un proyecto Laravel usando laravel new.
• Configuración de la conexión a base de datos MySQL en el archivo .env.
• Creación del modelo Estudiante y su migración.
• Uso de Artisan para ejecutar migraciones.
• Implementación de rutas y controlador EstudianteController.
• Creación de una vista con Blade y estilos con Tailwind CSS.
• Registro de estudiantes mediante formularios y persistencia en base de datos.

Reflexión

Aprendí que PHP sigue siendo una tecnología muy importante para el desarrollo web del lado servidor. Laravel facilita enormemente la organización del código, el acceso a bases de datos y la creación de aplicaciones seguras y escalables. El uso de migraciones, controladores y vistas Blade permite trabajar de forma ordenada y profesional, reduciendo errores y mejorando la productividad.
