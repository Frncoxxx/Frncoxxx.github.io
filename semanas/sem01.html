SEMANA 1: Fundamentos de la Tecnología Web

La primera clase trató sobre cómo funciona la web y cuáles son las tecnologías que la hacen posible. Se explicó la diferencia entre tecnología nativa y tecnología web, destacando que la web es abierta, accesible y basada en estándares internacionales del W3C. También se aprendió que toda aplicación se compone de dos partes:
• Frontend: lo visual e interactivo (HTML, CSS, JavaScript).
• Backend: la lógica, los datos y la conexión con el servidor.
Además, se revisó el DNS, sistema que traduce los nombres de dominio a direcciones IP para poder acceder a las páginas.

Lo esencial aprendido

• Cómo se estructura una página web.
• La función del DNS y los protocolos HTTP/HTTPS.
• El uso de GitHub para documentar el trabajo y guardar avances.

Reflexión:

Esta semana me ayudó a comprender que la web no solo se trata de diseño, sino de comunicación entre sistemas. Ver cómo cada lenguaje cumple un papel diferente me hizo valorar la importancia de los estándares y la práctica constante.

Conclusión:

Pude identificar los elementos esenciales de una aplicación web y aplicar herramientas modernas para su desarrollo. El aprendizaje fue principalmente práctico, lo que me permitió afianzar los conceptos básicos del desarrollo web y reconocer la importancia del trabajo colaborativo en GitHub.
