    SEMANA 2: HTML, CSS , Boosttrap, Tailwind

    Esta semana las exposiciones se centraron en los pilares del desarrollo web visual: HTML y CSS, junto con dos herramientas potentes para diseñar más rápido —Bootstrap y Tailwind CSS.

    HTML es la base: define la estructura de una página.
    Por ejemplo:

    Código <header> <h1>Mi Portafolio</h1> <nav> <a href="#proyectos">Proyectos</a> </nav> </header>

    CSS da vida al diseño, agregando color y estilo:
    Por ejemplo:

    Código header { background: #004aad; color: white; text-align: center; }

    Bootstrap simplifica todo con clases ya creadas, ideales para proyectos rápidos:
    Por ejemplo:

    Código <div class="container text-center"> <button class="btn btn-success">Aceptar</button> </div>

    Tailwind CSS, en cambio, te deja personalizar directamente en el HTML con clases utilitarias:
    Por ejemplo:

    Código <div class="bg-gray-200 p-4 rounded shadow-md"> <p class="text-blue-600 font-semibold">Ejemplo con Tailwind</p> </div>

    Reflexión:

    Me pareció interesante ver cómo se puede construir una web desde cero con solo unas etiquetas y luego mejorarla visualmente con estilos. Bootstrap facilita crear estructuras limpias sin tanto código, mientras que Tailwind da más control y personalización.

    Conclusión:

    Esta semana aprendí que HTML organiza, CSS embellece y los frameworks aceleran el diseño. Aplicar lo aprendido me permitió entender cómo se combinan la estructura, estilo y eficiencia en un proyecto web moderno.
