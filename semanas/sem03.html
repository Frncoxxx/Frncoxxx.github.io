SEMANA 3: Prácticas 1 y 2

Esta semana se realizaron dos prácticas enfocadas en el desarrollo de interfaces web completas.

Práctica 1: HTML y CSS puro

Se elaboraron páginas web usando solo HTML y CSS, aplicando formularios interactivos, animaciones y diseño responsive con flexbox y media queries.
Se creó una página principal con cards animados y enlaces a otras secciones. Esta práctica reforzó el manejo de estilos, estructura y animaciones sin librerías externas.

Práctica 2: HTML y Tailwind CSS

Se desarrolló un proyecto colaborativo en GitHub, utilizando Tailwind CSS para construir listas y cards con diseño moderno y adaptable.
Se aplicaron grid, flexbox, efectos hover y breakpoints para diferentes tamaños de pantalla. Esta práctica permitió experimentar con un framework actual y comprender la importancia del trabajo en equipo.

Lo más importante

• HTML: permitió definir la estructura principal del sitio con formularios, imágenes y botones.
• CSS: se utilizó para dar color, animaciones y diseño responsive con media queries.
• Tailwind CSS: simplificó la personalización de estilos directamente desde el HTML.
• GitHub: se empleó para trabajar en equipo, crear ramas y unir versiones del proyecto.

Enlaces de los ejercicios de laboratorio

Aprendí que con HTML y CSS se pueden crear estructuras sólidas, pero con Tailwind se puede optimizar el diseño de manera más rápida y ordenada. Trabajar en equipo con GitHub me enseñó a colaborar de forma eficiente y profesional.

Conclusión:

Fue una semana práctica y productiva, donde se unieron la creatividad visual y la organización técnica. Ahora entiendo mejor cómo se construyen las interfaces modernas que vemos cada día en la web.
