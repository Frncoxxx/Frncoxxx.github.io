SEMANA 4: JavaScript

Durante la cuarta semana del curso se realizaron exposiciones sobre JavaScript, abordando tanto su uso básico como sus aplicaciones avanzadas.
El objetivo fue entender cómo este lenguaje permite agregar vida e interactividad a las páginas web. JavaScript es un lenguaje interpretado que se ejecuta directamente en el navegador.
Permite crear acciones dinámicas y responder a eventos como clics, movimientos o escritura del usuario.

Código let nombre = prompt("¿Cómo te llamas?"); alert("Hola " + nombre + ", ¡bienvenido!");
Aplicaciones prácticas

Durante las exposiciones se mostraron ejemplos reales de:
• Eventos: cómo hacer que un botón ejecute una acción.
• Manipulación del DOM: cambiar texto, color o contenido.
• Funciones flecha: una forma moderna de simplificar código.
• Objetos y arreglos: estructuras para almacenar datos.

Reflexión:

Esta semana me permitió ver que JavaScript es el puente entre el diseño y la lógica.
Sin él, una página web solo muestra información; con él, responde, calcula y reacciona.
Fue una experiencia útil para entender cómo se combinan la creatividad y la programación.

Conclusión:

Las exposiciones me ayudaron a comprender la importancia de la lógica de programación dentro del desarrollo web.
Con JavaScript puedo controlar elementos, validar datos y personalizar la experiencia del usuario. Ahora entiendo que este lenguaje es la base de muchas tecnologías modernas como React, Angular o Node.js.
