    SEMANA 5: Práctica 3 y React

    En esta semana se desarrolló la Práctica 3 y se dio la introducción a React.
    Primero construimos una aplicación con HTML, CSS y JavaScript puro, y luego instalamos Node.js para crear nuestro primer proyecto React.

    Parte 1 – La práctica 3

    • Creamos una ruleta interactiva que selecciona nombres aleatorios al girar.
    • Los datos se cargaban desde un textarea, y el sistema los guardaba automáticamente con LocalStorage.
    • Se añadió un sorteo de equipos, generando grupos al azar con opciones personalizadas.
    • También se agregaron botones para descargar, copiar o reiniciar los resultados.

    Parte 2 – Instalación y creación de proyectos React

    Después de instalar Node.js, aprendimos tres formas de iniciar un proyecto con React:

    1. Create React App
    Código npx create-react-app miProyecto npm start
    2. Vite
    Código npm create vite@latest miProyecto npm install npm run dev
    3. Next.js (React + servidor)
    Código npx create-next-app miProyecto npm run dev

    Reflexión:

    Aprendí a integrar todas las herramientas vistas hasta ahora: HTML, CSS, JavaScript y React.
    Instalar Node.js fue un paso importante porque permite manejar dependencias y automatizar tareas. Ahora entiendo que React organiza mejor el código, lo hace más limpio y fácil de mantener.

    Conclusión:

    Esta semana fue clave para conectar el desarrollo clásico con el moderno. Pasamos de programar manualmente en JavaScript a usar herramientas profesionales como Vite y React, lo que marca el siguiente nivel en el desarrollo web.
